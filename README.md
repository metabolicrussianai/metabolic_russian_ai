## Чего в Phoenix нет

Чтобы не было разночтений — вот чего мы не делаем:

- Нет backpropagation
- Нет функции потерь и скалярной цели обучения
- Нет градиентного спуска (SGD, Adam и прочих)
- Нет RLHF, PPO, DPO, actor-critic
- Нет attention из трансформеров
- Нет fine-tuning, LoRA и адаптеров
- Нет replay buffer и experience replay
- Нет reward-модели

## Публичная демка

`metabolic_ai_demo.py` — обфусцированный воспроизводимый артефакт. Снаружи — те же интерфейсы, что у настоящего движка: `tick()`, `read_state()`, `inject_stimulus()`. Внутри — детерминированный поток на основе SHA-256.

Сделано намеренно:

- партнёры и ревьюеры видят форму рантайма,
- биологическая динамика остаётся закрытой,
- три метода — `reconfigure_weights`, `phase_transition`, `metabolic_step` — поднимают `NotImplementedError` со ссылкой на NDA-контакт.

```bash
python metabolic_ai_demo.py
```
